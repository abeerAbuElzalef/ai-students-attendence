# מערכת מעקב נוכחות תלמידים
# Student Attendance Tracking System

מערכת נוכחות מקיפה עם תמיכה בעברית, זיהוי חגים ישראליים, וייצוא לאקסל.

![Israeli Holidays](https://img.shields.io/badge/Holidays-Israeli%20Calendar-green)
![Export](https://img.shields.io/badge/Export-Excel-brightgreen)

## תכונות עיקריות

✅ **ניהול תלמידים** - הוספה, עריכה ומחיקה של תלמידים עם שמות בעברית  
✅ **לוח שנה אינטראקטיבי** - צפייה בנוכחות יומית עם ממשק נוח  
✅ **זיהוי חגים** - זיהוי אוטומטי של חגים ישראליים ושבתות  
✅ **סימון נוכחות** - נוכח, חיסור, איחור, או חיסור מוצדק  
✅ **סטטיסטיקות** - ניתוח נוכחות לפי חודש עם גרפים  
✅ **ייצוא לאקסל** - דוחות חודשיים ודוחות תלמיד שנתיים  
✅ **נייד** - ניתן להריץ בכל מקום עם מסד נתונים מקומי

## דרישות מקדימות

- [Node.js](https://nodejs.org/) גרסה 18 ומעלה
- npm (מגיע עם Node.js)

## התקנה

### שלב 1: הורדה והתקנה

```bash
# כנס לתיקיית הפרויקט
cd ai-student-attendence

# התקן את כל התלויות
npm install
cd backend && npm install
cd ../frontend && npm install
cd ..
```

### שלב 2: הפעלת המערכת

```bash
# הפעל את השרת והממשק יחד
npm start
```

או בנפרד:

```bash
# בטרמינל אחד - הפעל את השרת
cd backend
npm start

# בטרמינל שני - הפעל את הממשק
cd frontend
npm start
```

### שלב 3: פתח את המערכת

פתח את הדפדפן בכתובת: **http://localhost:3000**

## שימוש במערכת

### הוספת תלמידים

1. עבור ללשונית **"תלמידים"**
2. לחץ על **"הוסף תלמיד"** להוספה בודדת
3. או לחץ על **"הוספה מרובה"** להוספת רשימה שלמה

**פורמט להוספה מרובה:**
```
יוסי כהן, א׳1
שרה לוי, א׳1
דוד ישראלי, א׳2
רחל אברהם
```

### סימון נוכחות

1. עבור ללשונית **"לוח שנה"**
2. לחץ על תאריך ברצונך לסמן נוכחות
3. לכל תלמיד, בחר אחת מהאפשרויות:
   - ✓ נוכח (ירוק)
   - ✗ חיסור (אדום)
   - ⏰ איחור (כתום)
   - ⚠ חיסור מוצדק (סגול)
4. לחץ **"שמור"**

### ייצוא לאקסל

1. עבור ללשונית **"ייצוא"**
2. בחר דוח חודשי או דוח תלמיד שנתי
3. בחר חודש/שנה ולחץ הורד

## גיבוי והעברה

מסד הנתונים נשמר בקובץ `backend/attendance.db`. להעברת המערכת למחשב אחר:

1. העתק את כל תיקיית הפרויקט
2. ודא שקובץ `attendance.db` כלול
3. התקן תלויות והפעל

## פתרון בעיות

### השרת לא עולה
```bash
# בדוק שפורט 3001 פנוי
netstat -ano | findstr :3001

# נסה פורט אחר
set PORT=4000 && npm start
```

### שגיאות התקנה
```bash
# נקה ונסה שוב
rm -rf node_modules
npm cache clean --force
npm install
```

### הממשק לא נטען
- ודא שהשרת רץ על פורט 3001
- נסה לרענן את הדף
- בדוק את קונסול הדפדפן לשגיאות
